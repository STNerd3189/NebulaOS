# NebulaOS Installer Flow

## Overview
The NebulaOS installer experience should feel guided, modern, and tailored to creators and gamers.

## Steps
1. Welcome screen
2. Language and region selection
3. Disk selection
4. User account setup
5. Driver and audio profile selection
6. Optional mode selection: Gaming, Streaming, Studio
7. Final installation and reboot

## Recommended Profiles
- Gaming: enables GameMode, MangoHud, Steam launcher shortcuts, and sci-fi performance overlays
- Streaming: enables OBS Studio, capture tools, audio routing, and neon-themed creator overlays
- Studio: enables Ardour, Audacity, low-latency audio setup, and a sleek cyber-interface layout
