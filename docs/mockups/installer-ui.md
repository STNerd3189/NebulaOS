# NebulaOS Installer UI Mockup

## Layout
- Header: NebulaOS logo and step title
- Left panel: summary of selected options
- Right panel: current configuration screen
- Footer: Back / Next / Cancel actions

## Recommended Steps
1. Welcome
2. Region and keyboard
3. Disk and partitioning
4. User account
5. Driver and audio profile
6. Mode selection
7. Install and reboot

## Visual Style
- dark glass panels
- neon accent buttons
- animated progress bar
- creator-gamer friendly language
