# NebulaOS Sci-Fi Theme Guide

## Visual Identity
NebulaOS should feel like a futuristic command center for gaming, creation, and audio production.

## Design Language
- dark space backgrounds
- neon cyan, purple, and electric blue accents
- holographic UI panels
- subtle animated glow effects
- cyberpunk-inspired typography and iconography

## Theme Goals
- make the desktop feel immersive and advanced
- support both gaming and studio workflows
- keep the interface polished and readable
