# NebulaOS Installer Workflow

## Objective
Provide a guided, creator-friendly installation experience that supports gaming, streaming, and studio workflows.

## Workflow
1. Welcome and license acceptance
2. Language and keyboard selection
3. Disk selection and partitioning
4. User account setup
5. Driver selection and hardware detection
6. Audio profile selection
7. Optional mode selection
   - Gaming
   - Streaming
   - Studio
8. Install packages and finalize setup

## Mode Profiles
- Gaming: enables performance tuning, Steam, and GameMode
- Streaming: enables OBS Studio, capture tools, and audio routing
- Studio: enables low-latency audio, DAWs, and editing tools
