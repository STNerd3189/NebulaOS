# Nebula Plasma Theme Concept

## Visual Direction
A futuristic creator-gamer desktop theme built around deep space colors, neon gradients, and a premium dark UI.

## Palette
- Background: #060816
- Primary Accent: #5B6CFF
- Secondary Accent: #18D1FF
- Highlight: #A855F7
- Text: #E8F1FF

## Suggested Design Rules
- dark mode by default
- soft glow on panels and buttons
- high-contrast icons and text
- subtle animated highlights for active windows

## How to Apply
1. Copy the theme assets into your KDE Plasma theme folder.
2. Open System Settings > Appearance.
3. Select the Nebula Plasma theme and apply it.
