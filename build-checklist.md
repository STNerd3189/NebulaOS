# NebulaOS Build Checklist

## Preparation
- define the target hardware support range
- choose the base Arch mirror and installer strategy
- prepare a custom live ISO profile

## System Setup
- install the base system
- configure bootloader and networking
- add essential utilities and desktop packages

## Desktop and Theme
- install KDE Plasma
- apply branding assets and wallpaper
- create a polished first-run experience

## Software Integration
- install gaming tools
- install streaming and recording tools
- install video, image, and audio tools

## Testing
- test boot
- test audio input and output
- test GPU and driver setup
- test OBS and screen capture
- test Steam and game launchers

## Release Prep
- document installation steps
- publish troubleshooting notes
- build the final ISO
