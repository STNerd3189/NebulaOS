# NebulaOS Roadmap

## Phase 1: Vision and Branding
- Finalize the distro identity
- Create the logo and visual style
- Define the product positioning

## Phase 2: Base System Foundation
- Build a minimal Arch-based prototype
- Configure bootloader, networking, and core packages
- Validate hardware detection

## Phase 3: Desktop Experience
- Install KDE Plasma
- Create a polished first-run experience
- Add system utilities and a modern theme

## Phase 4: Gaming Stack
- Add Steam, Lutris, Heroic, Bottles, and Proton
- Install MangoHud and GameMode
- Validate performance and compatibility

## Phase 5: Creator Workflow
- Add OBS Studio, Kdenlive, Shotcut, Blender, Krita, and GIMP
- Set up screen capture and recording presets
- Add export presets for common platforms

## Phase 6: Audio Production
- Install PipeWire and WirePlumber
- Add Ardour, Audacity, LMMS, Carla, and MuseScore
- Configure low-latency audio and MIDI support

## Phase 7: Beta Testing
- Release a beta ISO
- Collect feedback from users
- Fix bugs and improve stability

## Phase 8: Stable Release
- Polish the installer and onboarding
- Publish documentation and support channels
- Release the first official NebulaOS version
